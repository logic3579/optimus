# P4 — assets Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Land an AWS-only, read-only cloud-asset discovery console on top of P0/P1/P2/P3. Background `robfig/cron` workers pull EC2 instances, VPCs (with subnets), and RDS databases per (CloudAccount, region) and persist snapshots in Postgres; FE exposes filterable list pages. A new `assets.Consumer` Go seam is reserved for P5/P6.

**Architecture:** New BE module `internal/modules/assets/` with sub-packages `account/`, `account/inuse/`, `instance/`, `vpc/`, `database/`, `awsclient/`, `sync/`, `sync/runs/`, plus the `consume.go` seam and `module/wire.go`. Per-account in-memory locks gate concurrent sweeps; authoritative full-page sweeps + `last_seen_at`-based soft-delete keep snapshots consistent without ever partial-deleting. Six new tables (`cloud_accounts`, `aws_instances`, `aws_vpcs`, `aws_subnets`, `aws_databases`, `assets_sync_runs`). Fifteen new `43xxx` error codes, five new `assets:*` permission codes. P1 cloudkey delete pre-checks an `assets/account/inuse.Counter`. New FE module `src/views/assets/` (lowercase/kebab) with five pages; no new direct FE deps.

**Tech Stack:** Go 1.25 + Gin + GORM + goose v3 (existing), `github.com/aws/aws-sdk-go-v2` + `config` + `credentials` + `service/ec2` + `service/rds` (NEW — pinned by Task 1), `github.com/robfig/cron/v3` v3.0.1 (NEW). Vue 3 + Ant Design Vue + Pinia + vue-router + `useTable`/`useI18n` hooks (all existing from P0–P3). No new FE direct deps.

**Reference spec:** `docs/superpowers/specs/2026-06-11-p4-assets-design.md` (commit `b4a68fd`).

## Invariants (read once, hold throughout)

These hold for the whole plan. If a step seems to contradict one, stop and re-read the spec before writing code.

1. **`credentials.Consumer` is the sole cloudkey path.** Never import `credentials/cloudkey` directly to decrypt. Always `consumer.GetCloudKey(ctx, id, purpose)` with `purpose` of the form `assets.sync.<reason>`. Defer `credentials.Wipe(ck)` on every successful fetch. Spec §4.1 / §5.2.

2. **Per-request AWS clients.** `awsclient.For(...)` is called fresh inside every sweep — never cached across sweeps or accounts. Mirrors P2 `k8s/client/factory.go`. Spec §5.2.

3. **Authoritative-only soft-delete.** A `(cloud_account, region, resource_type)` sweep soft-deletes ONLY when `fetcher.FetchAll(ctx)` returned `(items, nil)`. Any pagination error, ctx cancel, or AWS error → status `failed`, items_seen=0, items_softdeleted=0, no UPDATE…SET deleted_at issued. Spec §4.2 / §4.6.

4. **In-memory per-account lock.** `engine.tryLock(accountID)` gates both `cron` and `manual` paths. Manual handler returns 409 `CodeAssetsSyncBusy` without spawning. Cron tick that finds the lock held writes ONE `assets_sync_runs` row with status `skipped` for that account and moves on. Lock is released by `defer engine.unlock(accountID)` inside the worker. Spec §4.3.

5. **VPC + Subnet are one sweep unit.** Both upserted/soft-deleted in the SAME transaction. Either fetch failing aborts both. `assets_sync_runs.resource_type` value `network` denotes this joint sweep. Spec §4.2 / §3.6.

6. **Manual sync handler is async.** Handler does: load account → tryLock → write audit row → spawn goroutine with `context.WithoutCancel(c.Request.Context())` + ceiling timeout → return 200. Goroutine owns the lock and `defer`s `engine.unlock` as its FIRST line. The public `engine.RunAccount` is for tests only; the handler calls `engine.runAccountLocked(ctx, ...)` directly to skip a redundant lock acquisition. Spec §4.4.

7. **All comments in English.** Code in this repo, including comments, is English-only. Chinese is fine in PR descriptions / chat but never in `.go` / `.ts` / `.vue` / `.sql`. (P3 carry-forward feedback.)

8. **No raw error text to clients.** Every HTTP error envelopes through `apperr.New / apperr.Wrap`; AWS errors get normalized via `awsclient.MapError(err)` into the 43102-43107 range and stored on `assets_sync_runs.error_code` + `error`. Original `err` carries to slog via the `Wrap(cause, ...)` form. Spec §5.3.

9. **Audit on every mutation, but NOT on cron sweeps.** Manual sync writes one `assets.cloud_account.sync_trigger` audit row (synchronous, before goroutine spawn). cron sweeps write nothing to audit — observability lives in `assets_sync_runs`. `credentials.Consumer.GetCloudKey(ctx, id, "assets.sync")` produces its own `credentials.consume.cloud_key` row from P1's plumbing — that's expected; do not double-record. Spec §10.

10. **`go.mod` `go` directive must remain `1.25` after deps.** Adding AWS SDK or `robfig/cron` must not raise it. If `go mod tidy` shows `go 1.26` anywhere, pin offending modules. P2 client-go pin precedent. Spec §16.

11. **P1 cloudkey patch is nil-safe.** `cloudkey.Service.assetsAccountsInUse` is an interface field that may be `nil` (so P1 still compiles + tests run standalone). The check `if s.assetsAccountsInUse != nil { … }` guards every call site. `cmd/server/main.go` injects the real implementation after P1 wiring and before route mount.

12. **PUT enabled_regions shrinkage explicitly soft-deletes.** When PUT removes a region from `cloud_accounts.enabled_regions`, the service issues an explicit `UPDATE aws_instances/aws_vpcs/aws_subnets/aws_databases SET deleted_at=now() WHERE cloud_account_id=$ AND region=$removed AND deleted_at IS NULL` for every removed region (transactional with the account UPDATE). Without this, sweeps stop visiting the region and the rows linger forever. Spec §7.1.

13. **`assets.Consumer` does not audit and takes no purpose.** Resource lookups for P5/P6 are non-sensitive and high-frequency. The Consumer signature is `LookupInstanceByPrivateIP(ctx, ip)` etc., never `(..., purpose string)`. Spec §6.

14. **bun on the FE.** Never `npm` / `pnpm` / `yarn`. CI runs `bun install --frozen-lockfile`. (P0 carry-forward.)

15. **i18n parity.** Every key added to one locale must appear in the other. `bun run i18n:check` is wired into CI.

16. **Coverage gate ≥ 60% per `assets/*` package.** Matches P0/P1/P2/P3 standard.

17. **One task = one commit (intermediate commits allowed).** Each task ends with a HEREDOC commit. Multi-step tasks may carry intermediate commits where natural (DTO commit → service commit → handler commit) but each commit must compile and pass its own tests.

18. **FE view directory casing matters.** macOS APFS is case-insensitive, Linux CI/prod isn't. P4 assets dirs MUST be lowercase / kebab (`assets/cloud-accounts`, `assets/sync-runs`, etc.) to match seed `component:` strings. (P3 carry-forward feedback.)

19. **Colima docker socket.** Integration tests require Docker. On this workstation: `export DOCKER_HOST=unix:///Users/logic/.colima/docker.sock` and `colima start` before `make test-int`. Without it, dockertest hangs. (P3 carry-forward.)

20. **`apperr.Wrap(cause, code, key, msg)`** is the cause-preserving wrap helper. There is no `BizError.WithCause`. **`apperr.New(code, key, msg)`** takes three fixed string args — not variadic. (P3 carry-forward feedback.)

21. **`audit.Event` shape:** `{UserID *uint64, Action, TargetType, TargetID string, IP, UserAgent, Payload any}`. `TargetID` is STRING (use `strconv.FormatUint`). (P3 carry-forward.)

22. **Service mutations take `ip, ua string`** (audit plumbing). Copy the P3 `application.Service` pattern: `(ctx, actorID, ip, ua, req)`. Handlers pass `c.ClientIP()` + `c.GetHeader("User-Agent")`. (P3 carry-forward.)

23. **Handlers use inline `strconv.ParseUint` + `c.GetUint64(middleware.CtxKeyUserID)`** — no `middleware.PathUint64` / `MustActorID`. Handler `Create` returns 200 OK (envelope convention), not 201. (P3 carry-forward.)

24. **FE injects use string keys** (`inject<AssetsAccountApi>('assetsAccountApi')!`) — no `keys.ts` file. API modules are functional factories (`makeAssetsAccountApi(client)`), not classes. `useI18n` is from `@/hooks/useI18n`. `useTable<T,F>` from `@/hooks/useTable`. Locale JSON is **nested**, not flat. (P3 carry-forward.)

25. **Built-in roles are `admin` (wildcard `*`) + `viewer` (`%:read` LIKE).** No new built-in roles. Permission picks happen automatically at seed. (P3 carry-forward.)

### Pattern D — Fake AWS SDK client for fetcher tests

The AWS SDK v2 service clients are concrete structs but they accept an `Options.HTTPClient` (any `aws.HTTPClient`-shaped value) and an `Options.APIOptions` middleware stack. The cleanest way to test fetchers is to construct the SDK client with a stub HTTP transport:

```go
type stubTransport struct{ responses map[string]string } // key = "ServiceID.OperationName"

func (s *stubTransport) Do(r *http.Request) (*http.Response, error) {
    target := r.Header.Get("X-Amz-Target") // for JSON RPC services (RDS)
    op := r.URL.Query().Get("Action")      // for query services (EC2)
    key := op
    if target != "" { key = target }
    body, ok := s.responses[key]
    if !ok { return nil, fmt.Errorf("stub: no response for %s", key) }
    return &http.Response{
        StatusCode: 200, Body: io.NopCloser(strings.NewReader(body)),
        Header: http.Header{"Content-Type": []string{"text/xml"}},
    }, nil
}
```

For EC2 (XML query protocol), the body is the canonical AWS XML response. For RDS (also query protocol with XML responses), same shape. Capture real XML from AWS once (or from SDK fixtures) and inline as constants in `_test.go`. The stub transport is wired via `ec2.NewFromConfig(cfg, func(o *ec2.Options) { o.HTTPClient = &http.Client{Transport: stub} })`.

```go
import (
```

```go
var regionRegex = regexp.MustCompile(`^[a-z]{2}-[a-z]+-\d$`)

// CloudKeyExistenceChecker is satisfied by P1 cloudkey.Service (Exists method).
type CloudKeyExistenceChecker interface {
	Exists(ctx context.Context, id uint64) (bool, error)
}

type Service struct {
	repo  *Repo
	audit *audit.Recorder
	cks   CloudKeyExistenceChecker
}

func NewService(repo *Repo, rec *audit.Recorder, cks CloudKeyExistenceChecker) *Service {
	return &Service{repo: repo, audit: rec, cks: cks}
}

func (s *Service) validateRegions(regions []string) error {
	if len(regions) == 0 {
		return apperr.New(errs.CodeAssetsRegionInvalid, errs.KeyRegionInvalid, "at least one region required")
	}
	for _, r := range regions {
		if !regionRegex.MatchString(r) {
			return apperr.New(errs.CodeAssetsRegionInvalid, errs.KeyRegionInvalid, "invalid region: "+r)
		}
	}
	return nil
}

func (s *Service) Create(ctx context.Context, actorID uint64, ip, ua string, req CreateRequest) (*Detail, error) {
	if req.Provider != "aws" {
		return nil, apperr.New(errs.CodeAssetsProviderUnsupported, errs.KeyProviderUnsupported, "only aws supported")
	}
	if err := s.validateRegions(req.EnabledRegions); err != nil { return nil, err }
	ok, err := s.cks.Exists(ctx, req.CloudKeyID)
	if err != nil { return nil, err }
	if !ok {
		return nil, apperr.New(errs.CodeAssetsCloudKeyNotFound, errs.KeyCloudKeyNotFound, "cloudkey not found")
	}
	conflict, err := s.repo.FindNameAlive(ctx, req.Name, 0)
	if err != nil { return nil, err }
	if conflict {
		return nil, apperr.New(errs.CodeAssetsCloudAccountNameConflict, errs.KeyCloudAccountNameConflict, "name already exists")
	}
	enabled := true
	if req.Enabled != nil { enabled = *req.Enabled }
	row := &models.CloudAccount{
		Name: req.Name, Provider: req.Provider, CloudKeyID: req.CloudKeyID,
		EnabledRegions: models.StringArray(req.EnabledRegions),
		Enabled:        enabled, Description: req.Description,
	}
	if err := s.repo.Create(ctx, row); err != nil { return nil, err }
	s.writeAudit(ctx, actorIDOrNil(actorID), "assets.cloud_account.create", row.ID, ip, ua, map[string]any{
		"name": req.Name, "provider": req.Provider, "cloudkey_id": req.CloudKeyID, "regions": req.EnabledRegions,
	})
	return s.detail(ctx, row.ID)
}

func (s *Service) Get(ctx context.Context, id uint64) (*Detail, error) {
	return s.detail(ctx, id)
}

func (s *Service) Update(ctx context.Context, actorID uint64, ip, ua string, id uint64, req UpdateRequest) (*Detail, error) {
	row, err := s.repo.FindByID(ctx, id)
	if err != nil { return nil, err }
	changed := map[string]any{}
	regionsRemoved := []string{}
	if req.Name != nil && *req.Name != row.Name {
		conflict, err := s.repo.FindNameAlive(ctx, *req.Name, row.ID)
		if err != nil { return nil, err }
		if conflict {
			return nil, apperr.New(errs.CodeAssetsCloudAccountNameConflict, errs.KeyCloudAccountNameConflict, "name already exists")
		}
		row.Name = *req.Name
		changed["name"] = *req.Name
	}
	if req.EnabledRegions != nil {
		if err := s.validateRegions(req.EnabledRegions); err != nil { return nil, err }
		newSet := map[string]bool{}
		for _, r := range req.EnabledRegions { newSet[r] = true }
		for _, r := range row.EnabledRegions {
			if !newSet[r] { regionsRemoved = append(regionsRemoved, r) }
		}
		row.EnabledRegions = models.StringArray(req.EnabledRegions)
		changed["regions"] = req.EnabledRegions
	}
	if req.Enabled != nil && *req.Enabled != row.Enabled {
		row.Enabled = *req.Enabled
		changed["enabled"] = *req.Enabled
	}
	if req.Description != nil && *req.Description != row.Description {
		row.Description = *req.Description
		changed["description"] = *req.Description
	}
	if len(changed) == 0 {
		return s.detail(ctx, id)
	}
	err = s.repo.db.Transaction(func(tx *gorm.DB) error {
		if err := tx.Save(row).Error; err != nil { return err }
		if len(regionsRemoved) > 0 {
			if _, err := s.repo.CascadeSoftDeleteResources(ctx, tx, row.ID, regionsRemoved); err != nil {
				return err
			}
		}
		return nil
	})
	if err != nil { return nil, err }
	s.writeAudit(ctx, actorIDOrNil(actorID), "assets.cloud_account.update", row.ID, ip, ua, map[string]any{
		"changed_fields": keysOf(changed), "regions_removed": regionsRemoved,
	})
	return s.detail(ctx, id)
}

func (s *Service) Delete(ctx context.Context, actorID uint64, ip, ua string, id uint64) (int64, error) {
	row, err := s.repo.FindByID(ctx, id)
	if err != nil { return 0, err }
	var cascaded int64
	err = s.repo.db.Transaction(func(tx *gorm.DB) error {
		n, err := s.repo.CascadeSoftDeleteResources(ctx, tx, row.ID, nil)
		if err != nil { return err }
		cascaded = n
		return tx.Delete(&models.CloudAccount{}, id).Error
	})
	if err != nil { return 0, err }
	s.writeAudit(ctx, actorIDOrNil(actorID), "assets.cloud_account.delete", row.ID, ip, ua, map[string]any{
		"name": row.Name, "cascaded_resources_count": cascaded,
	})
	return cascaded, nil
}

func (s *Service) List(ctx context.Context, q ListQuery) (*ListResponse, error) {
	items, total, err := s.repo.List(ctx, q)
	if err != nil { return nil, err }
	ids := make([]uint64, 0, len(items))
	for _, it := range items { ids = append(ids, it.CloudKeyID) }
	names, _ := s.repo.CloudKeyNames(ctx, ids)
	for i := range items { items[i].CloudKeyName = names[items[i].CloudKeyID] }
	return &ListResponse{Items: items, Total: total}, nil
}

func (s *Service) detail(ctx context.Context, id uint64) (*Detail, error) {
	row, err := s.repo.FindByID(ctx, id)
	if err != nil { return nil, err }
	names, _ := s.repo.CloudKeyNames(ctx, []uint64{row.CloudKeyID})
	d := toDetail(*row, names[row.CloudKeyID], nil, "")
	return &d, nil
}

// RecordSyncTrigger writes the assets.cloud_account.sync_trigger audit row.
// Called synchronously by the manual-sync handler BEFORE spawning the
// detached sweep goroutine (spec §4.4 / §10).
func (s *Service) RecordSyncTrigger(ctx context.Context, actor *uint64, ip, ua string, accountID uint64) {
	row, _ := s.repo.FindByID(ctx, accountID)
	regions := []string{}
	if row != nil { regions = []string(row.EnabledRegions) }
	s.writeAudit(ctx, actor, "assets.cloud_account.sync_trigger", accountID, ip, ua, map[string]any{
		"regions": regions, "trigger": "manual",
	})
}

func (s *Service) writeAudit(ctx context.Context, actor *uint64, action string, id uint64, ip, ua string, payload map[string]any) {
	_ = s.audit.Record(ctx, audit.Event{
		UserID: actor, Action: action,
		TargetType: "cloud_account", TargetID: strconv.FormatUint(id, 10),
		IP: ip, UserAgent: ua, Payload: payload,
	})
}

func actorIDOrNil(id uint64) *uint64 { if id == 0 { return nil } ; return &id }

func keysOf(m map[string]any) []string {
	out := make([]string, 0, len(m))
	for k := range m {
		if k == "regions_removed" { continue }
		out = append(out, k)
	}
	return out
}
```

```go
package awsclient

import (
	"context"
	"errors"
	"net"

	"optimus-be/internal/modules/assets/errs"

	"github.com/aws/smithy-go"
)

// MapError normalises an AWS SDK error into a 43xxx code + message_key + human message.
// Callers (fetchers / scheduler) store the triple into assets_sync_runs.
// Returns (0, "", "") for nil input.
func MapError(err error) (code int, key string, msg string) {
	if err == nil {
		return 0, "", ""
	}
	if errors.Is(err, context.DeadlineExceeded) || errors.Is(err, context.Canceled) {
		return errs.CodeAssetsAWSUnreachable, errs.KeyAWSUnreachable, "request timed out or cancelled"
	}
	var ne net.Error
	if errors.As(err, &ne) {
		return errs.CodeAssetsAWSUnreachable, errs.KeyAWSUnreachable, ne.Error()
	}
	var apiErr smithy.APIError
	if errors.As(err, &apiErr) {
		switch apiErr.ErrorCode() {
		case "AuthFailure", "InvalidClientTokenId", "SignatureDoesNotMatch", "ExpiredToken":
			return errs.CodeAssetsAWSUnauthorized, errs.KeyAWSUnauthorized, apiErr.ErrorMessage()
		case "UnauthorizedOperation", "AccessDenied", "AccessDeniedException":
			return errs.CodeAssetsAWSForbidden, errs.KeyAWSForbidden, apiErr.ErrorMessage()
		case "RequestCanceled":
			return errs.CodeAssetsAWSUnreachable, errs.KeyAWSUnreachable, apiErr.ErrorMessage()
		case "Throttling", "ThrottlingException", "RequestLimitExceeded", "TooManyRequestsException":
			return errs.CodeAssetsAWSThrottled, errs.KeyAWSThrottled, apiErr.ErrorMessage()
		default:
			return errs.CodeAssetsAWSOther, errs.KeyAWSOther, apiErr.ErrorCode() + ": " + apiErr.ErrorMessage()
		}
	}
	return errs.CodeAssetsAWSOther, errs.KeyAWSOther, err.Error()
}
```

```go
import (
	"context"
	"net/http"
	"time"

	"optimus-be/internal/infra/apperr"
	"optimus-be/internal/modules/assets/errs"
	"optimus-be/internal/modules/credentials"

	"github.com/aws/aws-sdk-go-v2/service/ec2"
	"github.com/aws/aws-sdk-go-v2/service/rds"
)

// Clients bundles the AWS service clients a sync sweep needs.
// Constructed fresh per (cloudkey, region) sweep — never cached.
type Clients struct {
	EC2 *ec2.Client
	RDS *rds.Client
}

// For returns service clients for the supplied cloudkey + region.
// timeout caps every individual SDK HTTP request via http.Client.Timeout.
func For(ctx context.Context, ck *credentials.CloudKey, region string, timeout time.Duration) (*Clients, error) {
	if ck == nil || ck.Provider != "aws" {
		provider := "(nil)"
		if ck != nil { provider = ck.Provider }
		return nil, apperr.New(errs.CodeAssetsProviderUnsupported, errs.KeyProviderUnsupported,
			"provider not supported: "+provider)
	}
	cfg, err := awsconfig.LoadDefaultConfig(ctx,
		awsconfig.WithRegion(region),
		awsconfig.WithCredentialsProvider(
			awscreds.NewStaticCredentialsProvider(ck.AccessKeyID, ck.SecretAccessKey, ""),
		),
		awsconfig.WithRetryMaxAttempts(3),
		awsconfig.WithHTTPClient(&http.Client{Timeout: timeout}),
	)
	if err != nil {
		return nil, apperr.Wrap(err, errs.CodeAssetsAWSConfig, errs.KeyAWSConfig,
			"failed to build AWS SDK config")
	}
	return &Clients{
		EC2: ec2.NewFromConfig(cfg),
		RDS: rds.NewFromConfig(cfg),
	}, nil
}
```

```go
package runs

import (
	"context"
	"time"

	"optimus-be/internal/models"

	"gorm.io/gorm"
)

type Repo struct{ db *gorm.DB }

func NewRepo(db *gorm.DB) *Repo { return &Repo{db: db} }

type InsertRequest struct {
	CloudAccountID    uint64
	Region            string
	ResourceType      string // instance|network|database
	Trigger           string // cron|manual|test
	TriggeredByUserID *uint64
}

func (r *Repo) Insert(ctx context.Context, req InsertRequest) (uint64, error) {
	row := &models.AssetsSyncRun{
		CloudAccountID:    req.CloudAccountID,
		Region:            req.Region,
		ResourceType:      req.ResourceType,
		StartedAt:         time.Now(),
		Status:            "running",
		Trigger:           req.Trigger,
		TriggeredByUserID: req.TriggeredByUserID,
	}
	if err := r.db.WithContext(ctx).Create(row).Error; err != nil {
		return 0, err
	}
	return row.ID, nil
}

type FinishRequest struct {
	Status           string // success|failed|skipped
	ItemsSeen        int32
	ItemsSoftDeleted int32
	Error            string
	ErrorCode        int32
}

func (r *Repo) Finish(ctx context.Context, id uint64, req FinishRequest) error {
	now := time.Now()
	return r.db.WithContext(ctx).Model(&models.AssetsSyncRun{}).Where("id = ?", id).
		Updates(map[string]any{
			"finished_at":       &now,
			"status":            req.Status,
			"items_seen":        req.ItemsSeen,
			"items_softdeleted": req.ItemsSoftDeleted,
			"error":             req.Error,
			"error_code":        req.ErrorCode,
		}).Error
}

// Prune removes sync_runs older than `olderThan` days.
func (r *Repo) Prune(ctx context.Context, olderThanDays int) (int64, error) {
	if olderThanDays <= 0 { return 0, nil }
	cutoff := time.Now().AddDate(0, 0, -olderThanDays)
	res := r.db.WithContext(ctx).
		Where("started_at < ?", cutoff).
		Delete(&models.AssetsSyncRun{})
	return res.RowsAffected, res.Error
}
```

```go
// Package sync implements the assets sync engine — sweep loop, scheduler,
// and concrete fetchers for EC2/VPC/RDS.
//
// Sweep semantics: a (cloud_account, region, resource_type) sweep is
// "authoritative" only when fetcher.FetchAll returned nil error after
// paginating to completion. Authoritative sweeps upsert items AND soft-
// delete every row whose last_seen_at is older than the sweep_start
// timestamp. Non-authoritative sweeps update nothing.
package sync

import (
	"context"
	"time"

	"optimus-be/internal/models"

	"gorm.io/gorm"
	"gorm.io/gorm/clause"
)

// UpsertInstances writes the freshly-fetched items and soft-deletes any rows
// in (accountID, region) whose last_seen_at is older than sweepStart.
// Runs in a single transaction. Returns the soft-deleted row count.
func UpsertInstances(ctx context.Context, db *gorm.DB, accountID uint64, region string, sweepStart time.Time, items []models.AWSInstance) (int64, error) {
	var softdel int64
	err := db.WithContext(ctx).Transaction(func(tx *gorm.DB) error {
		for i := range items {
			items[i].LastSeenAt = sweepStart
		}
		if len(items) > 0 {
			if err := tx.Clauses(clause.OnConflict{
				Columns: []clause.Column{{Name: "cloud_account_id"}, {Name: "region"}, {Name: "instance_id"}},
				DoUpdates: clause.AssignmentColumns([]string{
					"name", "instance_type", "state", "private_ip", "public_ip",
					"vpc_id", "subnet_id", "availability_zone", "launch_time", "tags",
					"last_seen_at", "updated_at",
				}),
			}).Create(&items).Error; err != nil {
				return err
			}
			// Revive any soft-deleted that were re-upserted in this batch.
			if err := tx.Unscoped().
				Model(&models.AWSInstance{}).
				Where("cloud_account_id = ? AND region = ? AND last_seen_at = ? AND deleted_at IS NOT NULL",
					accountID, region, sweepStart).
				Update("deleted_at", nil).Error; err != nil {
				return err
			}
		}
		// Soft-delete rows missed by this authoritative sweep.
		res := tx.Model(&models.AWSInstance{}).
			Where("cloud_account_id = ? AND region = ? AND last_seen_at < ? AND deleted_at IS NULL",
				accountID, region, sweepStart).
			Update("deleted_at", time.Now())
		softdel = res.RowsAffected
		return res.Error
	})
	return softdel, err
}

// UpsertVPCsAndSubnets writes vpcs+subnets in the SAME transaction.
// Either failing rolls back both. Soft-delete sweeps run on both tables
// scoped to (accountID, region).
func UpsertVPCsAndSubnets(ctx context.Context, db *gorm.DB, accountID uint64, region string, sweepStart time.Time, vpcs []models.AWSVPC, subnets []models.AWSSubnet) (vpcSoftdel, subnetSoftdel int64, err error) {
	err = db.WithContext(ctx).Transaction(func(tx *gorm.DB) error {
		for i := range vpcs { vpcs[i].LastSeenAt = sweepStart }
		for i := range subnets { subnets[i].LastSeenAt = sweepStart }
		if len(vpcs) > 0 {
			if err := tx.Clauses(clause.OnConflict{
				Columns: []clause.Column{{Name: "cloud_account_id"}, {Name: "region"}, {Name: "vpc_id"}},
				DoUpdates: clause.AssignmentColumns([]string{
					"name", "cidr_block", "is_default", "state", "tags", "last_seen_at", "updated_at",
				}),
			}).Create(&vpcs).Error; err != nil { return err }
			if err := tx.Unscoped().Model(&models.AWSVPC{}).
				Where("cloud_account_id = ? AND region = ? AND last_seen_at = ? AND deleted_at IS NOT NULL",
					accountID, region, sweepStart).
				Update("deleted_at", nil).Error; err != nil { return err }
		}
		if len(subnets) > 0 {
			if err := tx.Clauses(clause.OnConflict{
				Columns: []clause.Column{{Name: "cloud_account_id"}, {Name: "region"}, {Name: "subnet_id"}},
				DoUpdates: clause.AssignmentColumns([]string{
					"vpc_id", "cidr_block", "availability_zone", "name", "tags",
					"last_seen_at", "updated_at",
				}),
			}).Create(&subnets).Error; err != nil { return err }
			if err := tx.Unscoped().Model(&models.AWSSubnet{}).
				Where("cloud_account_id = ? AND region = ? AND last_seen_at = ? AND deleted_at IS NOT NULL",
					accountID, region, sweepStart).
				Update("deleted_at", nil).Error; err != nil { return err }
		}
		now := time.Now()
		resV := tx.Model(&models.AWSVPC{}).
			Where("cloud_account_id = ? AND region = ? AND last_seen_at < ? AND deleted_at IS NULL",
				accountID, region, sweepStart).
			Update("deleted_at", now)
		if resV.Error != nil { return resV.Error }
		vpcSoftdel = resV.RowsAffected
		resS := tx.Model(&models.AWSSubnet{}).
			Where("cloud_account_id = ? AND region = ? AND last_seen_at < ? AND deleted_at IS NULL",
				accountID, region, sweepStart).
			Update("deleted_at", now)
		subnetSoftdel = resS.RowsAffected
		return resS.Error
	})
	return vpcSoftdel, subnetSoftdel, err
}

// UpsertDatabases writes RDS instances; symmetric to UpsertInstances.
func UpsertDatabases(ctx context.Context, db *gorm.DB, accountID uint64, region string, sweepStart time.Time, items []models.AWSDatabase) (int64, error) {
	var softdel int64
	err := db.WithContext(ctx).Transaction(func(tx *gorm.DB) error {
		for i := range items { items[i].LastSeenAt = sweepStart }
		if len(items) > 0 {
			if err := tx.Clauses(clause.OnConflict{
				Columns: []clause.Column{{Name: "cloud_account_id"}, {Name: "region"}, {Name: "db_instance_id"}},
				DoUpdates: clause.AssignmentColumns([]string{
					"engine", "engine_version", "instance_class", "status", "endpoint", "port",
					"multi_az", "publicly_accessible", "storage_gb", "tags",
					"last_seen_at", "updated_at",
				}),
			}).Create(&items).Error; err != nil { return err }
			if err := tx.Unscoped().Model(&models.AWSDatabase{}).
				Where("cloud_account_id = ? AND region = ? AND last_seen_at = ? AND deleted_at IS NOT NULL",
					accountID, region, sweepStart).
				Update("deleted_at", nil).Error; err != nil { return err }
		}
		res := tx.Model(&models.AWSDatabase{}).
			Where("cloud_account_id = ? AND region = ? AND last_seen_at < ? AND deleted_at IS NULL",
				accountID, region, sweepStart).
			Update("deleted_at", time.Now())
		softdel = res.RowsAffected
		return res.Error
	})
	return softdel, err
}
```

```go
import (
	"context"
	"sync"
	"time"

// Clients is the bundle returned by ClientFactory. Re-export so engine.go
// doesn't import awsclient at the public boundary (keeps the package
// substitutable in tests).
type Clients = awsclient.Clients

// ClientFactory builds AWS service clients for a (cloudkey, region) sweep.
// Production impl is awsclient.For; tests can pass a stub.
type ClientFactory interface {
	For(ctx context.Context, ck *credentials.CloudKey, region string, timeout time.Duration) (*Clients, error)
}

// CloudKeyConsumer is satisfied by credentials.Consumer. Narrow interface
// so tests can stub.
type CloudKeyConsumer interface {
	GetCloudKey(ctx context.Context, id uint64, purpose string) (*credentials.CloudKey, error)
}

// Fetcher abstracts the three resource fetchers. Real impls in
// ec2_fetcher.go / vpc_fetcher.go / rds_fetcher.go (Tasks 9-11).
type Fetcher interface {
	FetchInstances(ctx context.Context, c *Clients) ([]models.AWSInstance, error)
	FetchVPCsAndSubnets(ctx context.Context, c *Clients) ([]models.AWSVPC, []models.AWSSubnet, error)
	FetchDatabases(ctx context.Context, c *Clients) ([]models.AWSDatabase, error)
}

type Engine struct {
	db        *gorm.DB
	runs      *runs.Repo
	factory   ClientFactory
	fetcher   Fetcher
	consumer  CloudKeyConsumer
	timeout   time.Duration
	locks     sync.Map // map[uint64]bool
}

func NewEngine(db *gorm.DB, r *runs.Repo, f ClientFactory, fetcher Fetcher, c CloudKeyConsumer, awsTimeout time.Duration) *Engine {
	return &Engine{db: db, runs: r, factory: f, fetcher: fetcher, consumer: c, timeout: awsTimeout}
}

func (e *Engine) TryLock(accountID uint64) bool {
	_, loaded := e.locks.LoadOrStore(accountID, true)
	return !loaded
}

func (e *Engine) Unlock(accountID uint64) { e.locks.Delete(accountID) }

// RunAll iterates all enabled, non-deleted CloudAccounts and runs every
// region × type sweep that is due. trigger is "cron" or "test".
func (e *Engine) RunAll(ctx context.Context, trigger string) error {
	var accounts []models.CloudAccount
	if err := e.db.WithContext(ctx).
		Where("enabled = true AND deleted_at IS NULL").
		Find(&accounts).Error; err != nil {
		return err
	}
	for _, a := range accounts {
		_ = e.RunAccount(ctx, a.ID, trigger, nil) // per-account errors logged inside; loop never aborts
	}
	return nil
}

// RunAccount acquires the per-account lock then delegates to runAccountLocked.
// Public for tests; the manual-sync handler in account/handler.go bypasses
// this and calls runAccountLocked directly (because it pre-acquired the lock).
func (e *Engine) RunAccount(ctx context.Context, accountID uint64, trigger string, triggeredBy *uint64) error {
	if !e.TryLock(accountID) {
		// Mark one skipped row to make the cron decision visible in sync_runs history.
		var acct models.CloudAccount
		if err := e.db.WithContext(ctx).First(&acct, accountID).Error; err == nil {
			region := ""
			if len(acct.EnabledRegions) > 0 { region = acct.EnabledRegions[0] }
			runID, ierr := e.runs.Insert(ctx, runs.InsertRequest{
				CloudAccountID: accountID, Region: region, ResourceType: "instance",
				Trigger: trigger, TriggeredByUserID: triggeredBy,
			})
			if ierr == nil {
				_ = e.runs.Finish(ctx, runID, runs.FinishRequest{Status: "skipped"})
			}
		}
		return nil
	}
	defer e.Unlock(accountID)
	return e.runAccountLocked(ctx, accountID, trigger, triggeredBy)
}

// runAccountLocked is the production entry path used by the manual-sync handler
// (handler does its own TryLock + spawns goroutine + invokes this).
func (e *Engine) runAccountLocked(ctx context.Context, accountID uint64, trigger string, triggeredBy *uint64) error {
	var acct models.CloudAccount
	if err := e.db.WithContext(ctx).First(&acct, accountID).Error; err != nil { return err }
	ck, err := e.consumer.GetCloudKey(ctx, acct.CloudKeyID, "assets.sync")
	if err != nil { return err }
	defer credentials.Wipe(ck)

	for _, region := range acct.EnabledRegions {
		clients, cerr := e.factory.For(ctx, ck, region, e.timeout)
		if cerr != nil {
			e.recordFailed(ctx, accountID, region, "instance", trigger, triggeredBy, cerr)
			continue
		}
		e.sweepInstances(ctx, accountID, region, clients, trigger, triggeredBy)
		e.sweepNetwork(ctx, accountID, region, clients, trigger, triggeredBy)
		e.sweepDatabases(ctx, accountID, region, clients, trigger, triggeredBy)
	}
	return nil
}

func (e *Engine) sweepInstances(ctx context.Context, accountID uint64, region string, c *Clients, trigger string, triggeredBy *uint64) {
	runID, err := e.runs.Insert(ctx, runs.InsertRequest{
		CloudAccountID: accountID, Region: region, ResourceType: "instance",
		Trigger: trigger, TriggeredByUserID: triggeredBy,
	})
	if err != nil { return }
	sweepStart := time.Now()
	items, ferr := e.fetcher.FetchInstances(ctx, c)
	if ferr != nil {
		code, _, msg := awsclient.MapError(ferr)
		_ = e.runs.Finish(ctx, runID, runs.FinishRequest{Status: "failed", Error: msg, ErrorCode: int32(code)})
		return
	}
	for i := range items {
		items[i].CloudAccountID = accountID
		items[i].Region = region
	}
	softdel, uerr := UpsertInstances(ctx, e.db, accountID, region, sweepStart, items)
	if uerr != nil {
		_ = e.runs.Finish(ctx, runID, runs.FinishRequest{Status: "failed", Error: uerr.Error(), ErrorCode: 0})
		return
	}
	_ = e.runs.Finish(ctx, runID, runs.FinishRequest{
		Status: "success", ItemsSeen: int32(len(items)), ItemsSoftDeleted: int32(softdel),
	})
}

func (e *Engine) sweepNetwork(ctx context.Context, accountID uint64, region string, c *Clients, trigger string, triggeredBy *uint64) {
	runID, err := e.runs.Insert(ctx, runs.InsertRequest{
		CloudAccountID: accountID, Region: region, ResourceType: "network",
		Trigger: trigger, TriggeredByUserID: triggeredBy,
	})
	if err != nil { return }
	sweepStart := time.Now()
	vpcs, subnets, ferr := e.fetcher.FetchVPCsAndSubnets(ctx, c)
	if ferr != nil {
		code, _, msg := awsclient.MapError(ferr)
		_ = e.runs.Finish(ctx, runID, runs.FinishRequest{Status: "failed", Error: msg, ErrorCode: int32(code)})
		return
	}
	for i := range vpcs { vpcs[i].CloudAccountID = accountID ; vpcs[i].Region = region }
	for i := range subnets { subnets[i].CloudAccountID = accountID ; subnets[i].Region = region }
	vsd, ssd, uerr := UpsertVPCsAndSubnets(ctx, e.db, accountID, region, sweepStart, vpcs, subnets)
	if uerr != nil {
		_ = e.runs.Finish(ctx, runID, runs.FinishRequest{Status: "failed", Error: uerr.Error(), ErrorCode: 0})
		return
	}
	_ = e.runs.Finish(ctx, runID, runs.FinishRequest{
		Status: "success",
		ItemsSeen: int32(len(vpcs) + len(subnets)),
		ItemsSoftDeleted: int32(vsd + ssd),
	})
}

		CloudAccountID: accountID, Region: region, ResourceType: "database",

	if ferr != nil {
		code, _, msg := awsclient.MapError(ferr)
		_ = e.runs.Finish(ctx, runID, runs.FinishRequest{Status: "failed", Error: msg, ErrorCode: int32(code)})
		return
	}
	for i := range items { items[i].CloudAccountID = accountID ; items[i].Region = region }
	softdel, uerr := UpsertDatabases(ctx, e.db, accountID, region, sweepStart, items)
	if uerr != nil {
		_ = e.runs.Finish(ctx, runID, runs.FinishRequest{Status: "failed", Error: uerr.Error(), ErrorCode: 0})
		return
	}
	_ = e.runs.Finish(ctx, runID, runs.FinishRequest{
		Status: "success", ItemsSeen: int32(len(items)), ItemsSoftDeleted: int32(softdel),
	})
}

func (e *Engine) recordFailed(ctx context.Context, accountID uint64, region, resourceType, trigger string, triggeredBy *uint64, err error) {
	code, _, msg := awsclient.MapError(err)
	runID, ierr := e.runs.Insert(ctx, runs.InsertRequest{
		CloudAccountID: accountID, Region: region, ResourceType: resourceType,
		Trigger: trigger, TriggeredByUserID: triggeredBy,
	})
	if ierr != nil { return }
	_ = e.runs.Finish(ctx, runID, runs.FinishRequest{Status: "failed", Error: msg, ErrorCode: int32(code)})
}

// PruneSyncRuns removes runs older than `days`. Called by the daily 03:00 UTC
// cron entry registered by scheduler.go.
func (e *Engine) PruneSyncRuns(ctx context.Context, days int) (int64, error) {
	return e.runs.Prune(ctx, days)
}
```

```go
package sync

import (
	"context"
	"log/slog"
	"time"

	"github.com/robfig/cron/v3"
)

// Config is the assets cron config block loaded from YAML/env.
type Config struct {
	SyncCron           string        // e.g. "*/15 * * * *"
	StartupDelay       time.Duration // e.g. 30s
	RetentionDays      int           // e.g. 90
	AWSRequestTimeout  time.Duration // mirror: engine.timeout
}

// StartScheduler registers cron entries and runs them in a goroutine.
// Returns the *cron.Cron so callers can Stop() during shutdown.
func StartScheduler(ctx context.Context, cfg Config, engine *Engine, logger *slog.Logger) *cron.Cron {
	c := cron.New()
	time.AfterFunc(cfg.StartupDelay, func() {
		if _, err := c.AddFunc(cfg.SyncCron, func() {
			if err := engine.RunAll(context.Background(), "cron"); err != nil {
				logger.Error("assets.sync.cron.runall.err", "err", err)
			}
		}); err != nil {
			logger.Error("assets.sync.cron.register.err", "err", err)
			return
		}
		if _, err := c.AddFunc("0 3 * * *", func() {
			n, err := engine.PruneSyncRuns(context.Background(), cfg.RetentionDays)
			if err != nil {
				logger.Error("assets.sync.prune.err", "err", err)
				return
			}
			logger.Info("assets.sync.prune.ok", "deleted", n)
		}); err != nil {
			logger.Error("assets.sync.prune.register.err", "err", err)
		}
		c.Start()
		logger.Info("assets.sync.scheduler.started", "cron", cfg.SyncCron)
	})
	go func() {
		<-ctx.Done()
		stop := c.Stop()
		select {
		case <-stop.Done():
		case <-time.After(30 * time.Second):
			logger.Warn("assets.sync.scheduler.shutdown.timeout")
		}
	}()
	return c
}
```

```go
func (r *Repo) List(ctx context.Context, q ListQuery) ([]Summary, int64, error) {
	tx := r.db.WithContext(ctx).Table("assets_sync_runs")
	if q.AccountID != 0 { tx = tx.Where("cloud_account_id = ?", q.AccountID) }
	if q.ResourceType != "" { tx = tx.Where("resource_type = ?", q.ResourceType) }
	if q.Status != "" { tx = tx.Where("status = ?", q.Status) }
	if q.StartedAfter != "" {
		ts, err := time.Parse(time.RFC3339, q.StartedAfter)
		if err == nil { tx = tx.Where("started_at >= ?", ts) }
	}
	var total int64
	if err := tx.Count(&total).Error; err != nil { return nil, 0, err }

	var rows []struct {
		models.AssetsSyncRun
		AccountName string `gorm:"column:account_name"`
	}
	err := tx.Select("assets_sync_runs.*, cloud_accounts.name AS account_name").
		Joins("LEFT JOIN cloud_accounts ON cloud_accounts.id = assets_sync_runs.cloud_account_id").
		Order("assets_sync_runs.started_at DESC").
		Offset((q.Page - 1) * q.Size).
		Limit(q.Size).
		Scan(&rows).Error
	if err != nil { return nil, 0, err }

	out := make([]Summary, len(rows))
	for i, r := range rows {
		out[i] = Summary{
```

```go
package instance

import (
	"context"
	"encoding/json"

	"optimus-be/internal/models"

	"gorm.io/gorm"
)

type Repo struct{ db *gorm.DB }

func NewRepo(db *gorm.DB) *Repo { return &Repo{db: db} }

func (r *Repo) List(ctx context.Context, q ListQuery) ([]Summary, int64, error) {
	tx := r.db.WithContext(ctx).Table("aws_instances")
	if !q.IncludeDeleted {
		tx = tx.Where("aws_instances.deleted_at IS NULL")
	} else {
		tx = tx.Unscoped()
	}
	if q.AccountID != 0 { tx = tx.Where("aws_instances.cloud_account_id = ?", q.AccountID) }
	if q.Region != "" { tx = tx.Where("aws_instances.region = ?", q.Region) }
	if q.State != "" { tx = tx.Where("aws_instances.state = ?", q.State) }
	if q.VPCID != "" { tx = tx.Where("aws_instances.vpc_id = ?", q.VPCID) }
	if q.Q != "" {
		like := "%" + q.Q + "%"
		tx = tx.Where("aws_instances.name ILIKE ? OR aws_instances.instance_id ILIKE ? OR host(aws_instances.private_ip) ILIKE ? OR aws_instances.tags::text ILIKE ?",
			like, like, like, like)
	}
	var total int64
	if err := tx.Count(&total).Error; err != nil { return nil, 0, err }

	var rows []struct {
		models.AWSInstance
		AccountName string `gorm:"column:account_name"`
	}
	err := tx.Select("aws_instances.*, cloud_accounts.name AS account_name").
		Joins("LEFT JOIN cloud_accounts ON cloud_accounts.id = aws_instances.cloud_account_id").
		Order("aws_instances.deleted_at IS NULL DESC, aws_instances.last_seen_at DESC").
		Offset((q.Page - 1) * q.Size).
		Limit(q.Size).
		Scan(&rows).Error
	if err != nil { return nil, 0, err }

	out := make([]Summary, len(rows))
	for i, r := range rows { out[i] = toSummary(r.AWSInstance, r.AccountName) }
	return out, total, nil
}

func toSummary(m models.AWSInstance, accountName string) Summary {
	s := Summary{
		ID: m.ID, CloudAccountID: m.CloudAccountID, CloudAccountName: accountName,
		Region: m.Region, InstanceID: m.InstanceID, Name: m.Name,
		InstanceType: m.InstanceType, State: m.State,
		VPCID: m.VPCID, SubnetID: m.SubnetID, AvailabilityZone: m.AvailabilityZone,
		LaunchTime: m.LaunchTime, LastSeenAt: m.LastSeenAt, Deleted: m.DeletedAt.Valid,
	}
	if m.PrivateIP != nil { s.PrivateIP = m.PrivateIP.String() }
	if m.PublicIP != nil { s.PublicIP = m.PublicIP.String() }
	tags := map[string]string{}
	_ = json.Unmarshal(m.Tags, &tags)
	s.Tags = tags
	return s
}
```

```go
// Package assets exposes a Consumer seam so P5/P6 can look up resource
// snapshots without going through HTTP. No purpose parameter, no audit
// — resource lookups are non-sensitive and high-frequency by design.
package assets

import (
	"context"
	"errors"
	"net/netip"

	"optimus-be/internal/infra/apperr"
	"optimus-be/internal/models"
	"optimus-be/internal/modules/assets/errs"

	"gorm.io/gorm"
)

// Instance is the read-side view of an EC2 instance row.
type Instance struct {
	AccountID    int64
	AccountName  string
	Region       string
	InstanceID   string
	Name         string
	InstanceType string
	State        string
	PrivateIP    netip.Addr
	PublicIP     netip.Addr
	VPCID        string
	SubnetID     string
}

// ErrAssetsInstanceNotFound is returned by LookupInstanceBy* when no row matches.
var ErrAssetsInstanceNotFound = errors.New("assets: instance not found")

type Consumer interface {
	LookupInstanceByPrivateIP(ctx context.Context, ip netip.Addr) (*Instance, error)
	LookupInstanceByID(ctx context.Context, accountID int64, region, instanceID string) (*Instance, error)
	ListInstancesByVPC(ctx context.Context, accountID int64, region, vpcID string) ([]Instance, error)
}

type consumer struct{ db *gorm.DB }

func NewConsumer(db *gorm.DB) Consumer { return &consumer{db: db} }

func (c *consumer) LookupInstanceByPrivateIP(ctx context.Context, ip netip.Addr) (*Instance, error) {
	if !ip.IsValid() {
		return nil, apperr.New(0, "", "invalid IP address")
	}
	row, accountName, err := c.findOne(ctx, "aws_instances.private_ip = ?", ip.String())
	if err != nil { return nil, err }
	return toInstance(row, accountName), nil
}

func (c *consumer) LookupInstanceByID(ctx context.Context, accountID int64, region, instanceID string) (*Instance, error) {
	row, accountName, err := c.findOne(ctx,
		"aws_instances.cloud_account_id = ? AND aws_instances.region = ? AND aws_instances.instance_id = ?",
		accountID, region, instanceID)
	if err != nil { return nil, err }
	return toInstance(row, accountName), nil
}

func (c *consumer) ListInstancesByVPC(ctx context.Context, accountID int64, region, vpcID string) ([]Instance, error) {
	var rows []struct {
		models.AWSInstance
		AccountName string `gorm:"column:account_name"`
	}
	err := c.db.WithContext(ctx).Table("aws_instances").
		Select("aws_instances.*, cloud_accounts.name AS account_name").
		Joins("LEFT JOIN cloud_accounts ON cloud_accounts.id = aws_instances.cloud_account_id").
		Where("aws_instances.cloud_account_id = ? AND aws_instances.region = ? AND aws_instances.vpc_id = ? AND aws_instances.deleted_at IS NULL",
			accountID, region, vpcID).
		Order("aws_instances.last_seen_at DESC").
		Scan(&rows).Error
	if err != nil { return nil, err }
	out := make([]Instance, len(rows))
	for i, r := range rows { out[i] = *toInstance(r.AWSInstance, r.AccountName) }
	return out, nil
}

func (c *consumer) findOne(ctx context.Context, where string, args ...any) (models.AWSInstance, string, error) {
	var row struct {
		models.AWSInstance
		AccountName string `gorm:"column:account_name"`
	}
	err := c.db.WithContext(ctx).Table("aws_instances").
		Select("aws_instances.*, cloud_accounts.name AS account_name").
		Joins("LEFT JOIN cloud_accounts ON cloud_accounts.id = aws_instances.cloud_account_id").
		Where("aws_instances.deleted_at IS NULL").
		Where(where, args...).
		Order("aws_instances.last_seen_at DESC").
		Limit(1).
		Scan(&row).Error
	if err != nil { return models.AWSInstance{}, "", err }
	if row.ID == 0 {
		// Use a sentinel + concrete error code so callers can switch
		// regardless of whether they import errs.
		_ = errs.CodeAssetsCloudAccountNotFound // silence unused import in some build paths
		return models.AWSInstance{}, "", ErrAssetsInstanceNotFound
	}
	return row.AWSInstance, row.AccountName, nil
}

func toInstance(m models.AWSInstance, accountName string) *Instance {
	out := &Instance{
		AccountID: int64(m.CloudAccountID), AccountName: accountName,
		Region: m.Region, InstanceID: m.InstanceID, Name: m.Name,
		InstanceType: m.InstanceType, State: m.State,
		VPCID: m.VPCID, SubnetID: m.SubnetID,
	}
	if m.PrivateIP != nil { a, _ := netip.ParseAddr(m.PrivateIP.String()) ; out.PrivateIP = a }
	if m.PublicIP != nil { a, _ := netip.ParseAddr(m.PublicIP.String()) ; out.PublicIP = a }
	return out
}
```

```go
package module

import (

	credCloudKey "optimus-be/internal/modules/credentials/cloudkey"

	"github.com/gin-gonic/gin"
	"github.com/robfig/cron/v3"

// Wire builds the assets module and returns the public Consumer + the
// account inuse.Counter that P1 cloudkey.Service needs.
//
// Wire MUST run AFTER the P1 cloudkey module is constructed but BEFORE
// route mount; the composition root injects the Counter into cloudkey
// via SetAccountsInUseCounter before main.Run.
type Output struct {
	Consumer      assets.Consumer
	InUseCounter  inuse.Counter
	Engine        *sync.Engine
	CronScheduler *cron.Cron
}

type Input struct {
	DB                *gorm.DB
	Cfg               config.AssetsConfig
	Audit             *audit.Recorder
	CredentialsConsumer credentials.Consumer
	CloudKeyService   *credCloudKey.Service // for Exists() check
	PermissionCache   *rbac.PermissionCache
	Logger            *slog.Logger
}

func Wire(rootCtx context.Context, in Input) Output {
	// Account vertical
	accountRepo := account.NewRepo(in.DB)
	accountSvc := account.NewService(accountRepo, in.Audit, in.CloudKeyService) // CloudKeyService implements Exists
	accountInUse := inuse.New(in.DB)

	// Sync vertical
	runsRepo := runs.NewRepo(in.DB)
	runsSvc := runs.NewService(runsRepo)
	composite := &sync.CompositeFetcher{
		Instance: sync.EC2Fetcher{},
		Network:  sync.VPCFetcher{},
		Database: sync.RDSFetcher{},
	}
	type cfAdapter struct{}
	factory := &awsclient_factoryAdapter{} // see helper below
	engine := sync.NewEngine(in.DB, runsRepo, factory, composite, in.CredentialsConsumer, in.Cfg.AWSRequestTimeout)

	// Read-only resource verticals
	instanceSvc := instance.NewService(instance.NewRepo(in.DB))
	vpcSvc := vpc.NewService(vpc.NewRepo(in.DB))
	databaseSvc := database.NewService(database.NewRepo(in.DB))

	// Account handler needs an injected TriggerSync impl that calls engine.
	accountHandler := account.NewHandler(accountSvc)
	accountHandler.SetTriggerSync(func(c *gin.Context, id uint64) {
		// async path per spec §4.4
		// 1. account validation + lock acquisition + audit happen inline (synchronously)
		// 2. detached goroutine owns the lock and runs the sweep
		ctx := c.Request.Context()
		acct, err := accountSvc.Get(ctx, id)
		if err != nil { writeError(c, err); return }
		if !acct.Enabled {
			writeBizErr(c, errsCodeAssetsCloudAccountDisabled())
			return
		}
		if !engine.TryLock(id) {
			writeBizErr(c, errsCodeAssetsSyncBusy())
			return
		}
		userID := c.GetUint64(middleware.CtxKeyUserID)
		uidPtr := &userID
		// Synchronous audit before spawning the goroutine
		accountSvc.RecordSyncTrigger(ctx, &userID, c.ClientIP(), c.GetHeader("User-Agent"), id)
		go func() {
			defer engine.Unlock(id)
			detached := context.WithoutCancel(ctx)
			detached, cancel := context.WithTimeout(detached,
				in.Cfg.AWSRequestTimeout*time.Duration(len(acct.EnabledRegions))*3)
			defer cancel()
			_ = engineRunAccountLocked(detached, engine, id, "manual", uidPtr)
		}()
		writeJSON200(c, map[string]any{"queued": true, "started_at": time.Now()})
	})

	// Resource handlers
	instanceHandler := instance.NewHandler(instanceSvc)
	vpcHandler := vpc.NewHandler(vpcSvc)
	databaseHandler := database.NewHandler(databaseSvc)
	syncRunsHandler := runs.NewHandler(runsSvc)

	// Cron
	scheduler := sync.StartScheduler(rootCtx, sync.Config{
		SyncCron:          in.Cfg.SyncCron,
		StartupDelay:      in.Cfg.SyncStartupDelay,
		RetentionDays:     in.Cfg.SyncRunRetentionDays,
		AWSRequestTimeout: in.Cfg.AWSRequestTimeout,
	}, engine, in.Logger)

	return Output{
		Consumer:      assets.NewConsumer(in.DB),
		InUseCounter:  accountInUse,
		Engine:        engine,
		CronScheduler: scheduler,
	}
	_ = accountHandler
	_ = instanceHandler
	_ = vpcHandler
	_ = databaseHandler
	_ = syncRunsHandler
}

// Mount registers all assets routes onto the `/api/v1/assets` group with
// per-permission sub-groups.
func Mount(grp *gin.RouterGroup, in Input, out Output, handlers Handlers) {
	requirePerm := func(code string) gin.HandlerFunc {
		return middleware.RequirePermission(in.PermissionCache, code)
	}
	handlers.Account.Mount(grp, requirePerm)
	handlers.Instance.Mount(grp, requirePerm)
	handlers.VPC.Mount(grp, requirePerm)
	handlers.Database.Mount(grp, requirePerm)
	handlers.SyncRuns.Mount(grp, requirePerm)
}

// Handlers bundles the mounted-handler structs so main.go can construct
// once and mount once. (Or fold into Output if preferred.)
type Handlers struct {
	Account  *account.Handler
	Instance *instance.Handler
	VPC      *vpc.Handler
	Database *database.Handler
	SyncRuns *runs.Handler
}
```

```typescript
import type {
  CloudAccountDetail,

export interface AssetsAccountApi {
  list(params: { q?: string; provider?: string; enabled?: boolean; include_deleted?: boolean; page?: number; size?: number }): Promise<ListResponse<CloudAccountSummary>>
  get(id: number): Promise<CloudAccountDetail>
  create(req: CreateCloudAccountRequest): Promise<CloudAccountDetail>
  update(id: number, req: UpdateCloudAccountRequest): Promise<CloudAccountDetail>
  remove(id: number): Promise<{ cascaded_resources_count: number }>
  triggerSync(id: number): Promise<{ queued: boolean; started_at: string }>
}

export function makeAssetsAccountApi(client: AxiosInstance): AssetsAccountApi {
  const base = '/api/v1/assets/cloud-accounts'
  return {
    list: (params) => client.get(base, { params }).then((r) => r.data.data),
    get: (id) => client.get(`${base}/${id}`).then((r) => r.data.data),
    create: (req) => client.post(base, req).then((r) => r.data.data),
    update: (id, req) => client.put(`${base}/${id}`, req).then((r) => r.data.data),
    remove: (id) => client.delete(`${base}/${id}`).then((r) => r.data.data),
    triggerSync: (id) => client.post(`${base}/${id}/sync`).then((r) => r.data.data),
```

```markdown
**`assets` module** (`internal/modules/assets/`): AWS-only read-only cloud-resource discovery console. `cloud_accounts` is the user-CRUD entity binding `(P1 cloudkey, enabled_regions[])`. `aws_instances` / `aws_vpcs`(+`aws_subnets`) / `aws_databases` are sweep-populated snapshots, soft-deleted via `last_seen_at` only on **authoritative** sweeps (every page returned `err == nil`).
```
